GateWay 文档

网关概述

springboot 2.0之前的版本会使用zuul，2.0之后会选用gateway作为新一代的网关组件，
  • 针对请求统一鉴权，限流，熔断，日志
  • 协议转化，针对后端多种不同的协议，在网关层统一处理后以HTTP或HTTPS协议统一对外暴露
  • 统一错误码处理，所有的请求返回值统一包装为标准格式
  • 请求转发，基于gateway网关实现内外网隔离，所有的外部请求通过网关转发到内部

网关作用

  • 性能 保证API高可用，请求负载机制，容错降级机制
  • 安全 权限校验，敏感信息脱敏，黑白名单，后端链路签名
  • 日志 日志记录（spanid，traceid）一旦涉及分布式，全链路跟踪必不可少，那么就需要日志收集
  • 缓存 数据缓存
  • 监控 记录请求响应数据，api耗时统计，性能分析
  • 限流 流量控制，错峰流控，内置多种限流规则
  • 灰度 线上灰度发布，减少风险
  • 路由 动态路由，实时刷新

微服务对网关的要求

  • 稳定性 作为所有微服务的总入口和请求分发的中转站，那么必须要求极高的稳定性
  • 安全性 防止恶意请求刷接口，保障数据传输的安全性
  • 高性能，高可用 网关的性能要求非常高，不能成为微服务的性能瓶颈，不然微服务将没有性能可言
  • 扩展性，维护性 对于定制化方面，内置了很多现成的解决方案，也可以参照进行自定义配置

GateWay配置分析

前面分析了HandlerMapping和WebHandle的扩展，知道了predicate是怎么加入到HandlerMapping来进行断言，也知道了在执行
exchange时的filterChain的运行，那么最后来看看整体是怎么组装起来运行的，通过配置文件看看，各个bean是怎么注册组装配合的
public class GatewayAutoConfiguration {
    ....
    // 注册RouteLocator bena ，route定位器有了，用于组装routes,将收集所有的predicate和filter
    @Bean
    public RouteLocator routeDefinitionRouteLocator(GatewayProperties properties,
            // 所有实现GatewayFilterFactory接口的bean都将被注入
            List<GatewayFilterFactory> gatewayFilters,
            // 所有实现RoutePredicateFactory接口的bean都将被注入
            List<RoutePredicateFactory> predicates,
            RouteDefinitionLocator routeDefinitionLocator,
            ConfigurationService configurationService) {
        return new RouteDefinitionRouteLocator(routeDefinitionLocator, predicates,
                gatewayFilters, properties, configurationService);
    }
    // FilteringWebHandler bean ，filterChain切入了
    @Bean
    public FilteringWebHandler filteringWebHandler(List<GlobalFilter> globalFilters) {
        return new FilteringWebHandler(globalFilters);
    }
    // 注册RoutePredicateHandlerMapping bean ，predicate的执行有了
    @Bean
    public RoutePredicateHandlerMapping routePredicateHandlerMapping(
            FilteringWebHandler webHandler, RouteLocator routeLocator,
            GlobalCorsProperties globalCorsProperties, Environment environment) {
        return new RoutePredicateHandlerMapping(webHandler, routeLocator,
                globalCorsProperties, environment);
    }
    ...
}
